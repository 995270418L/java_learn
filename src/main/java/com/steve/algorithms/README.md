* 策略：
    1. 继续leetcode前300道题中的normal题

    4. 关注复杂度分析，根据答案和解题过程，多思考加深对复杂度的理解

* 目标:
    1. 对题目找到思路和对应的算法，并实现代码（不必限制时间，不必 bug free）
